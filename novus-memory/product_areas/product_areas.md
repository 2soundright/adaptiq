# Product Areas

Key product areas of AdaptIQ with their features and capabilities.

| NAME | DESCRIPTION |
|------|-------------|
| [Authentication](authentication.md) | User login and account registration flow. Serves as the entry point to the application, supporting role-based access (user, worker, admin) with email/password authentication. |
| [Chat](chat.md) | The core AI assistant chat interface where users ask questions and receive AI-generated answers powered by a RAG pipeline. Includes source citations, conversation history, and response feedback. |
| [Document Management](document_management.md) | Admin tool for uploading and managing knowledge base documents. Documents are parsed, chunked, embedded, and indexed to power the AI assistant's retrieval pipeline. |
| [Analytics](analytics.md) | Admin dashboard for monitoring AI assistant usage, user satisfaction, top questions, document performance, and continual learning health metrics. |
| [Audit Logs](audit_logs.md) | Admin security and compliance tool showing encrypted audit log entries, including safety-blocked queries and toxic content flags. |
| [Web Scraper](web_scraper.md) | Admin tool for managing automated web content ingestion. Crawls the configured website, extracts content, and indexes it into the knowledge base for the AI assistant. |
